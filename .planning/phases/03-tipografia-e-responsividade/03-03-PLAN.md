**INSERÇÃO 4 — JS do hambúrguer (adicionar dentro do `&lt;script&gt;` existente, APÓS o código de parallax):**

```javascript
// Hambúrguer menu
const hamburger = document.querySelector('.nav-hamburger');
const navOverlay = document.getElementById('navOverlay');
const overlayLinks = navOverlay.querySelectorAll('a');

function openMenu() {
  hamburger.classList.add('open');
  navOverlay.classList.add('open');
  hamburger.setAttribute('aria-expanded', 'true');
  navOverlay.setAttribute('aria-hidden', 'false');
  document.body.style.overflow = 'hidden';
}

function closeMenu() {
  hamburger.classList.remove('open');
  navOverlay.classList.remove('open');
  hamburger.setAttribute('aria-expanded', 'false');
  navOverlay.setAttribute('aria-hidden', 'true');
  document.body.style.overflow = '';
}

hamburger.addEventListener('click', () => {
  hamburger.classList.contains('open') ? closeMenu() : openMenu();
});

overlayLinks.forEach(link => {
  link.addEventListener('click', (e) => {
    e.preventDefault();
    const target = document.querySelector(link.getAttribute('href'));
    closeMenu();
    setTimeout(() => {
      target.scrollIntoView({ behavior: 'smooth' });
    }, 350);
  });
});
```

NÃO usar transform:scale() no overlay — causa blur em texto no iOS.
NÃO usar height:0→auto — transition não funciona em height:auto.
NÃO adicionar position:fixed no body — causa layout jump no iOS. Usar overflow:hidden.

Adicionar regra CSS dentro do bloco `@media(max-width:700px)` existente no index.html:

```css
  #contato .form-g input,
  #contato .form-g textarea,
  #contato .form-g select {
    font-size: 16px;
  }
```

Razão: iOS Safari aciona zoom automático ao focar em qualquer input com font-size menor que 16px. O valor atual dos inputs é 13px (herdado do body). 16px é o limiar exato do iOS — qualquer valor abaixo aciona o zoom. NÃO usar `user-scalable=no` no viewport meta — isso bloqueia o zoom intencional do usuário e quebra acessibilidade.

Verificar que o seletor atinge os inputs: os campos de nome, email, tel e a textarea de mensagem estão todos dentro de `#contato .form-g`. O `select` é incluído por precaução, mesmo que não exista atualmente no form.
